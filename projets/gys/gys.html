Aide au service après-vente de GYS

Élaboration d'un outil de pré-diagnostic à partir des retours de pannes

Contexte

Ce projet est réalisé dans le cadre d'un challenge proposé dans le cadre du West Data Festival, et de porte sur les données de retours de panne de l'entreprise Gys. Cette entreprise est spécialisée dans la fabrication de chargeurs de batteries et possède un bureau d'études avec des ingénieurs dans différentes spécialités pour concevoir ses produits. Les produits sont fabriqués dans différents ateliers, et sont distribués dans le monde entier. En cas de panne, les produits sont renvoyés à Gys pour être analysés par des techniciens formés par l'entreprise. Le but du challenge est de travailler sur ces données de retours de panne pour créer un outil d'aide au diagnostic. L'objectif est d'accélérer la réparation des produits en proposant aux clients un pré-diagnostic à faire avant de renvoyer le produit. Un jeu de données avec tout l'historique a été fourni pour le challenge et toutes les idées pour utiliser ces données sont les bienvenues.

Travail réalisé

Pour avons décidé de construire nous même la base de données, car cela nous permettait de bien connaitre la composition du dataset, de choisir nous même les genres à inclure dans notre modèle. Nous avons utilisé des plateformes de streaming telles que Youtube et Shazam.

Une implémentation de l'algorithme Random Forest de zéro

Pour avons décidé de construire nous même la base données, car cela nous permettait de bien connaitre la composition du dataset, de choisir nous même les genres à inclure dans notre modèle.

Le genre musical est un ensemble de morceaux de musique, de formes musicales possédant plusieurs caractéristiques communes. Notre objectif est de développer un modèle de machine learning capable de reconnaître les genres d’une musique, et d’attribuer pour chaque genre, un pourcentage d’appartenance. Notre projet permet ainsi de connaître quels sont les mouvements musicaux, quelles inspirations ont influencé l’auteur de la musique lors de son écriture. De plus, connaître les genres d’une musique est une base solide pour construire un algorithme de recommandation.
