Introduction

Le genre musical est un ensemble de morceaux de musique, de formes musicales possédant plusieurs caractéristiques communes. Notre objectif est de développer un modèle de machine learning capable de reconnaître les genres d’une musique, et d’attribuer pour chaque genre, un pourcentage d’appartenance. Notre projet permet ainsi de connaître quels sont les mouvements musicaux, quelles inspirations ont influencé l’auteur de la musique lors de son écriture. De plus, connaître les genres d’une musique est une base solide pour construire un algorithme de recommandation.

Constitution de la base de données

Pour avons décidé de construire nous même la base de données, car cela nous permettait de bien connaitre la composition du dataset, de choisir nous même les genres à inclure dans notre modèle. Nous avons utilisé des plateformes de streaming telles que Youtube et Shazam.
